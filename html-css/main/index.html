Semantic HTML

 
  div is a generic container. 
  Instead of using div - semantic HTML
  encourages the use of descriptive containers.
  For example:
  -> header,
  -> nav,
  -> article,
  -> figure,
  -> footer, etc.
  These containers do not act differently,
  but they're supposed to work better for SEO.
